# HalDms

This is an unofficial client library for the
[Hal Leonard Digital Asset Management](https://haldms.halleonard.com/vendoradmin/)
system.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hal_dms'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hal_dms
